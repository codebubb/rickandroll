Create Short URLs to Rick Roll your friends.

Instructions

You can generate a short URL to send to your friends; they'll be redirected to a Rick Roll video of your choice and then you can view stats of where it was accessed from!

  • Generate a short URL using the form above.
  • Forward the disguised, short url link to your colleagues by whatever means you can.
  • View the stats by adding /viewstats to the end of your short URL ! (for example https://51w.site/7pdgUd/viewstats

You can generate as many short URLs as you like.

About

This is a simple URL shortener app that was made as part of a full, end-to-end project on the Junior Developer Central YouTube channel.

Although its a simple app, the idea of the project was to create something from scratch going through all the processes involved in a project from the design, to purchasing the domain(s) and hosting and configuring a build pipeline to automatically build and push the built code to a production server.

Whilst there are many different (and easier!) ways of setting up an environment like this, the idea of the granular tutorials is to go in depth with all the details of how it might be done with the available tools and then you can decide to replace any parts of the system if you find a different way of doing something.
